# VfW FFmpeg Bridge (for TrackMania and other software)
This project aims to provide an interface for legacy VfW applications to allow them to export video into modern formats using modern software and hardware encoders provided by FFmpeg.

Specifically this project is optimized for the **Shoot Video** feature in the **TrackMania** franchise, for which it also provides several other improvements - most notably the option to Export Audio without needing to have Stereo Mix set up as the default microphone system-wide.
